Q1. Which factors appear to influence subscription renewal the most?
The most influential factors for subscription renewal were:
Age
Watch Hours Per Week
Ad Clicks
Devices Used
Subscription Type (to a lesser extent)
These variables showed the highest importance in the renewal prediction model, suggesting that user engagement and activity strongly affect whether a customer renews their subscription.

Q2. Why is subscription renewal a classification problem?
Subscription renewal is a classification problem because the target variable has two categories:
Yes (customer renews)
No (customer does not renew)
The goal is to predict which class a customer belongs to, making it a binary classification task.

Q3. Why is monthly spending a regression problem?
Monthly spending is a regression problem because the target variable is a continuous numerical value (e.g., ₹353, ₹804, ₹317).
Instead of predicting categories, the model predicts an exact amount of money spent by the customer.
